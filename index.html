proper string spacing

Howdy. This is a string spacing calculator for musical instrument design based on Doug Proper's method. While the intention is to design the nut of a guitar, you could design a piano agraffe or a harp bridge with this if you wanted. Please input values in inches, with resolution down to thousandths of an inch. The default is for 6-strings, but you can add/remove as many strings as you want. If you wanted to do a 12-string, do each course independently. The outputs will be the offset distance of each string from the first string in inches.
